Generate YouTube Data API Key

  1. Open Google Cloud Console.
  2. Create or select a project.
  3. Enable YouTube Data API v3.
  4. Open Credentials and click Create Credentials, then API key.
  5. Copy key and optionally restrict it for safety.
  6. Paste it in this app and fetch videos.

Security tip: Restrict the key by HTTP referrer and YouTube Data API v3 scope.
