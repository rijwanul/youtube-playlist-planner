# Youtube Playlist Planner

**Youtube Playlist Planner for Study, Research or Exploration**

Plan, track, and finish YouTube playlists efficiently for learning, research, or deep exploration.

## Features
- Add one or more YouTube playlists or videos (comma separated)
- Track watched/unwatched videos
- Manual, normal, and reverse sequence modes
- Drag-and-drop manual reordering
- Motivational progress and ETA
- Import/export your progress (optionally with API key)
- In-app YouTube player with custom controls

## Usage
1. Paste playlist or video links/IDs and your YouTube Data API key.
2. Click "Fetch Video(s)" to load videos.
3. Mark videos as completed as you watch.
4. Use sequence controls to reorder or drag manually.
5. Export or import your progress as needed.

## Requirements
- YouTube Data API v3 key (see in-app help for how to get one)
- Modern browser
